# \<uprtcl-common>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i uprtcl-common
```

## Usage
```html
<script type="module">
  import 'uprtcl-common/uprtcl-common.js';
</script>

<uprtcl-common></uprtcl-common>
```

```bash
npm run test
```
